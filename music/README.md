# Музыка для роликов

Положи сюда **лицензионные / роялти-фри** mp3-треки.

- `default.mp3` — трек по умолчанию (подкладывается, если в запросе не передан `music_url`).
- Можно добавить несколько треков и выбирать их по названию из бэкенда (передавать `music_url`).

⚠️ Используй только треки с правом на коммерческое использование (CC0 / выкупленная лицензия),
иначе площадки (TikTok/YouTube) заблокируют контент за музыку.

Источники роялти-фри: Pixabay Music, Uppbeat, Epidemic Sound (платно), YouTube Audio Library.
